Lab 18: Simple NodeJS MVC App

This is a list of examples. See the app.js file to see what happens when these links are clicked.

  1. Example 1
    1. Going to the domain:your_port/ sends a request to app.js
    2. which routes the request to the router.get('/') function in controller/index.js
    3. which returns the static index.html file to the user
  2. Example 2 - /hello routes to index.js and sends back a plain text message
  3. Example 3 - /template_example routes to index.js and sends back this home page, but using a template.
  4. Example 4 - /student/create, routes requests to router.get('student/create') in controller/student. An ajax form for creating a student is returned.
  5. Example 5 - /student/all displays all users in a table
  6. Example 6 - /student/dropdown displays all student in a drop down menu
